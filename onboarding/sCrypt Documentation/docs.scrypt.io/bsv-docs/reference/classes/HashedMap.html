HashedMap

scrypt-ts / HashedMap

Class: HashedMap<K, V>

The HashedMap library provides a map/hashtable-like data structure. Unique keys and their corresponding values are hashed before being stored. Only the hash values of key and value are saved on the chain. HashedMap is a wrapper for Map in javascript. Only some of the specified functions can be called in the @method function of a contract, but there is no restriction on other places.
